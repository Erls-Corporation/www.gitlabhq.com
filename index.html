Project hosting/managment application for your servers using Git.

Your own lite app for projects/repositories hosting on your server. Fast, secure and stable solution based on ruby on rails.

Includes source-code browser, issues, comments.

You can brose projects, files, commits, etc. Manage project team. Create issues, comment on them.
View commits diff, browse commit code. Add public key to your account with one click. Download file from specified commit.
